# Real-Time Data Collection and Visualization Platform

## Overview
A real-time data collection and visualization platform built with React frontend, backend API (Flask/Node.js), WebSockets/SSE for real-time communication, database, and Power BI dashboards.

## Project Structure
- `frontend/` - React UI, real-time frontend updates, API integration (Web Developer)
- `backend/` - API and real-time services (Shared/API + Data Science Engineer for analytics)
- `analytics/` - Data modeling, Power BI dashboards, schemas (Data Science Engineer)
  - `powerbi/` - Power BI files and configurations
  - `schemas/` - Data schemas and models
  - `metrics.md` - Analytics metrics documentation
- `docs/` - Documentation
- `README.md` - This file

## Team Responsibilities
- **Web Developer**: Frontend development
- **Data Science Engineer**: Backend analytics, data modeling, Power BI
